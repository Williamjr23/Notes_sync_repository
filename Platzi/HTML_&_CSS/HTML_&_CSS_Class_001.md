# Anatomía de un elemento HTML, atributos, anidamiento y elementos vacíos (conceptos iniciales de html)


### Anatomía de un elemento HTML:

Este es un ejemplo de elemento HTML:
```
<h1> hola </h1>
```

Y se compone de:

Etiqueta de apertura:
```
<h1>
```

Contenido:
```
hola
```

Etiqueta de cierre: 
```
</h1>
```
